# House Price Prediction using Machine Learning

A Machine Learning project that predicts residential house prices using historical housing data. This project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), feature engineering, model training using Linear Regression, and model evaluation.

---

## Project Overview

The goal of this project is to build a regression model that can estimate the selling price of a house based on its physical features, location, and construction details.

The project follows a real-world machine learning pipeline, from understanding the dataset to evaluating the performance of the trained model.

---

## Objectives

- Understand the housing dataset
- Perform data cleaning and preprocessing
- Conduct Exploratory Data Analysis (EDA)
- Prepare data for machine learning
- Train a Linear Regression model
- Evaluate model performance using regression metrics
- Identify the key factors influencing house prices

---

## Dataset Features

The dataset contains information about residential properties, including:

- House Type
- Zoning Classification
- Lot Area
- Lot Configuration
- Building Type
- Overall Condition
- Construction Year
- Remodel Year
- Exterior Material
- Basement Area
- Sale Price (Target Variable)
